profile

This function starts a profiling scope in the current execution stack with a given description. It returns a Profile structure and measure elapsed time between this method invocation and Profile structure drop. It supports nested profiling scopes in case when this function invoked multiple times at the execution stack. In this case the profiling information will be nested at the output. Profiling information is being printed in the stderr.
